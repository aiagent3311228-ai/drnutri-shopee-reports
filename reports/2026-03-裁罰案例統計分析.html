🔴 紅燈：必罰詞（碰到就罰）

  • 治療 / 治癒 / 根治 + 任何疾病名
  • 預防 + 癌症 / 糖尿病
  • 殺癌 / 消滅癌細胞
  • 溶脂 / 碎脂 / 崩脂 / 排脂
  • 視神經修復 / 再生
  • 軟骨再生 / 骨細胞再生
  • 天然胰島素
  • 瘦瘦針原理
  • 外泌體 / 幹細胞激活
  • 具體 % 數字 + 改善疾病
  • N 天瘦 N 公斤

🟠 橘燈：高危詞（極易觸法）

  • 改善 + 疾病名稱
  • 修復 / 逆轉 + 身體組織
  • 降低（膽固醇/血糖/血脂）
  • 提升免疫力
  • 消除（病菌/感染）
  • 醫師推薦 / 臨床證實
  • 美白 / 淡斑
  • 壯陽 / 勃起
  • 解酒 / 排毒
  • 排宿便 / 清腸
  • 長高 / 增高
  • 豐胸 / 升CUP

🟡 黃燈：易踩線詞（需改寫）

  • 有感 / 見效（暗示療效）
  • N 天/週有感（時間+效果）
  • 使用前後對照
  • 個人見證 + 具體數字
  • 精準 / 標靶（暗示醫療）
  • 回春 / 凍齡
  • 護肝 / 養肝
  • 助眠 / 舒眠
  • 抗氧化（搭配誇大效果時）
  • N 倍吸收 / N 倍效果
🎯 判斷公式：「危險動詞（治療/改善/預防/修復）」＋「身體部位或疾病名稱」＋「具體數字（%/天/公斤）」＝ 三要素任取二者以上即構成違規。三者齊備則為重罰。
